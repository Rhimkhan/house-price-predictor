# Data Folder

Place the Kaggle House Prices dataset files here:

| File | Source |
|---|---|
| `train.csv` | [Kaggle Competition](https://www.kaggle.com/c/house-prices-advanced-regression-techniques/data) → Download `train.csv` |
| `test.csv`  | [Kaggle Competition](https://www.kaggle.com/c/house-prices-advanced-regression-techniques/data) → Download `test.csv`  |

## Quick Download Steps

1. Go to https://www.kaggle.com/c/house-prices-advanced-regression-techniques/data
2. Click **Download All**
3. Extract the ZIP and copy `train.csv` and `test.csv` into this `data/` folder
4. Then run: `python src/model_training.py`
